Sobre o GitHub

o GitHub é um serviço baseado em nuvem para versionamento e, além disso, uma rede social para que os desenvolvedores possam compartilhar seus conhecimentos e resoluções, contribuindo com projetos de outros Dev's e apresentando os seus para que outros possam contribuir, ou estudá-los.

Alguns comandos do Bash

Comando para criar um novo repositório
Git Init
Comitar informando Mensagem
git commit meuarquivo.txt -m "mensagem de commit"
Histórico de Commits
git log
Adicionar README
git add README
Criando um servidor remoto
git remote add orgin https://github.com/bruno9800/newrep.git
Enviar o repositório local para o remoto
git push -u origin master
